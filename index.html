한국인 취약 발음 가이드

L / R

가장 많이 틀리는 발음

L은 혀끝을 윗니 뒤에 붙이고, R은 혀를 어디에도 닿지 않게 뒤로 당기세요.

light / right lead / read liver / river
F / P

입술 실수

F는 윗니를 아랫입술에 살짝 대고 바람을 내세요. P는 두 입술을 붙였다가 터트립니다.

fan / pan fine / pine feel / peel
TH

혀를 내밀어야 해요

혀끝을 윗니와 아랫니 사이에 살짝 끼우고 바람을 내세요. 절대 '드/스'로 발음하면 안 됩니다.

think / sink that / dat three / tree
V / B

V는 진동이 느껴져야

V는 F와 같은 위치지만 성대를 울립니다. 아랫입술을 살짝 깨물어 보세요.

very / berry vest / best veil / bail
