1서비스 개요

CODEPRON(이하 "서비스")은 브라우저 기반의 AI 영어 발음 교정 서비스입니다. 사용자는 마이크를 통해 영어를 발화하고, 서비스는 Web Speech API 기반의 음성 인식을 통해 발음을 분석하여 피드백을 제공합니다.

본 서비스는 다음과 같은 기능을 제공합니다.

  • 난이도별(입문·중급·고급) 영어 발음 연습 문장 제공
  • 사용자 맞춤 스크립트 기반 발음 연습
  • 한국인이 자주 어려워하는 L/R, F/P, TH, V/B 발음 가이드
  • 브라우저 로컬 저장소 기반의 학습 기록 관리

2이용 조건

서비스를 이용하기 위한 기본 조건은 다음과 같습니다.

  • 연령 제한: 만 14세 이상 이용 가능합니다. 만 14세 미만은 법정대리인(부모 또는 보호자)의 동의가 필요합니다.
  • 브라우저 요건: Web Speech API를 지원하는 최신 브라우저(Google Chrome, Microsoft Edge, Safari 권장)를 사용해야 합니다.
  • 마이크 권한: 발음 분석 기능 이용을 위해 브라우저의 마이크 접근 허용이 필요합니다.
  • 인터넷 연결: 음성 인식 처리를 위해 인터넷 연결이 필요합니다.

3서비스 이용 범위

CODEPRON은 영어 발음 학습을 지원하는 교육적 도구입니다. 서비스 이용과 관련하여 다음 사항을 안내합니다.

  • 발음 점수 정확도: 제공되는 발음 점수는 브라우저의 음성 인식 결과를 기반으로 한 참고 지표이며, 절대적인 정확도를 보장하지 않습니다.
  • 학습 효과 보장 불가: CODEPRON은 영어 발음 향상을 돕는 도구이나, 특정 학습 성과를 보장하지 않습니다.
  • 무료 제공: 현재 CODEPRON의 핵심 기능은 무료로 제공됩니다. 향후 유료 기능이 추가될 수 있으며, 이 경우 사전에 공지합니다.
  • 서비스 변경: 서비스 내용, 기능, UI는 사전 예고 없이 변경될 수 있습니다.

4이용자 의무

서비스를 이용하는 사용자는 다음 사항을 준수해야 합니다.

  • 타인의 지적 재산권, 개인정보, 명예를 침해하는 행위 금지
  • 서비스를 악의적이거나 불법적인 목적으로 사용하는 행위 금지
  • 서비스의 정상적인 운영을 방해하는 행위(크롤링, 과부하 유발 등) 금지
  • 서비스 코드, 디자인, 콘텐츠를 무단으로 복제·배포·수정하는 행위 금지

위 의무를 위반하는 경우, 서비스 이용이 제한될 수 있습니다.

5지적 재산권

CODEPRON 서비스에 포함된 모든 콘텐츠(텍스트, 디자인, 로고, UI, 발음 예문, 코드 등)에 대한 지적 재산권은 CODEPRON에 귀속됩니다.

사용자는 개인적, 비상업적 학습 목적으로만 서비스를 이용할 수 있으며, 사전 서면 동의 없이 서비스 콘텐츠를 상업적으로 사용하거나 재배포할 수 없습니다.

6면책 조항

CODEPRON은 서비스 이용으로 발생하는 직접적·간접적 손해에 대해 법적 책임을 지지 않습니다.

다음과 같은 상황에서 CODEPRON은 책임을 지지 않습니다.

  • 브라우저 음성 인식 오류로 인한 발음 분석 부정확
  • 서비스 중단, 장애, 데이터 손실
  • 제3자 서비스(Google 음성 인식, AdSense 등) 관련 문제
  • 사용자의 기기, 네트워크, 브라우저 환경으로 인한 서비스 이상
  • 로컬 저장소 데이터 손실(브라우저 데이터 삭제, 기기 초기화 등)

7서비스 종료 및 변경

CODEPRON은 필요에 따라 서비스를 일시 중단하거나 종료할 수 있습니다. 서비스 종료 시에는 사전에 공지하며, 로컬 저장소에 저장된 사용자의 학습 데이터는 사용자가 직접 관리하시기 바랍니다.

CODEPRON은 서비스 운영상 필요에 따라 본 약관을 수정할 수 있습니다. 약관 변경 시 이 페이지에 업데이트된 내용을 게시하며, 중요한 변경의 경우 서비스 첫 화면을 통해 공지합니다. 변경된 약관의 적용 이후에도 서비스를 계속 이용하면 변경 사항에 동의하는 것으로 간주됩니다.
