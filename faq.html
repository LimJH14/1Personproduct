서비스 기본

CODEPRON은 한국인을 위한 브라우저 기반 AI 영어 발음 교정 서비스입니다. 마이크로 영어 문장을 읽으면 AI가 실시간으로 발음을 분석해 100점 만점 점수와 단어별 피드백을 제공합니다.

앱 설치나 회원가입 없이 Chrome, Edge 브라우저에서 바로 사용할 수 있으며, 모든 핵심 기능이 완전 무료입니다.

네, 완전히 무료입니다. 현재 CODEPRON의 발음 인식, 점수 계산, 발음 가이드, 스크립트 연습, 연습 기록 등 모든 기능을 비용 없이 사용할 수 있습니다.

서비스 유지를 위해 Google AdSense 광고가 일부 페이지에 표시될 수 있으나, 광고 클릭은 선택 사항이며 발음 연습 기능과 무관합니다.

아니요. 회원가입 없이 바로 사용할 수 있습니다. 연습 기록은 사용자의 브라우저 로컬 저장소(LocalStorage)에 저장되며, 서버로 전송되지 않습니다.

단, 다른 기기에서는 기록이 공유되지 않습니다. 기기 간 기록 동기화 기능은 향후 업데이트에서 제공될 예정입니다.

기술 및 작동 방식

브라우저에 내장된 Web Speech API를 사용합니다. 마이크 입력을 Google의 음성 인식 서버로 전송해 텍스트로 변환한 뒤, CODEPRON이 목표 문장과 비교 분석합니다.

변환된 텍스트는 레벤슈타인 거리(Levenshtein Distance) 알고리즘으로 처리되어 단어 일치율(60%)과 문자 유사도(40%)를 가중 평균한 발음 점수를 계산합니다.

CODEPRON의 점수는 음성 인식 결과와 목표 문장의 텍스트 유사도를 기반으로 합니다. 실제 발음의 음향적 특성(음높이, 억양, 연음 등)을 직접 분석하지는 않습니다.

이는 브라우저 기반의 특성상 한계이며, 그럼에도 "AI가 내 발음을 어떻게 인식하는가"를 확인하는 데 매우 유용합니다. 원어민이 이해하는 수준의 발음을 목표로 연습하기에 적합합니다.

더 정밀한 음소(phoneme) 수준 분석은 향후 업데이트에서 제공할 예정입니다.

음성 인식 기능은 Google Chrome(최신)과 Microsoft Edge(최신)에서 완전히 지원됩니다. 최상의 경험을 위해 Chrome 사용을 강력히 권장합니다.

Safari는 일부 버전에서 제한적으로 지원하며, Firefox는 Web Speech API를 지원하지 않아 음성 인식 기능을 사용할 수 없습니다. 단, 발음 가이드와 원어민 발음 듣기는 모든 브라우저에서 이용 가능합니다.

반응형 디자인으로 모바일 화면에 최적화되어 있습니다. Android 기기의 Chrome 브라우저에서 음성 인식이 지원됩니다.

iOS(iPhone/iPad)의 경우 Safari에서 Web Speech API 지원이 제한적입니다. iOS에서 사용 시 일부 기능이 정상 작동하지 않을 수 있습니다.

개인정보 및 보안

음성 데이터 자체는 CODEPRON 서버에 저장되지 않습니다. 음성 인식을 위해 브라우저가 Google의 Web Speech API 서버와 통신하지만, 이는 브라우저가 직접 처리하며 CODEPRON은 음성 파일에 접근하지 않습니다.

인식된 텍스트 결과와 연습 기록은 사용자 기기의 로컬 저장소에만 보관됩니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

음성 인식 기능 사용을 위해 브라우저 마이크 권한이 필요합니다. 마이크 버튼을 처음 클릭하면 브라우저가 권한을 요청합니다.

권한을 거부하면 발음 인식 기능을 사용할 수 없지만, 발음 가이드 읽기, 원어민 발음 듣기 등 다른 기능은 정상 이용 가능합니다. 마이크는 녹음 버튼을 누른 동안만 활성화됩니다.

발음 연습 활용법

낮은 점수에는 여러 원인이 있을 수 있습니다:

1. 발음 문제: 목표 발음과 실제 발음의 차이. 발음 가이드를 먼저 읽고 각 소리의 조음 방법을 이해한 뒤 연습하세요.

2. 말하는 속도: 너무 빠르거나 불명확하면 인식률이 떨어집니다. 천천히, 또렷하게 발음하세요.

3. 환경 소음: 조용한 환경에서 연습하세요. 배경 소음이 많으면 인식 정확도가 낮아집니다.

4. 마이크 품질: 이어폰 마이크나 헤드셋이 내장 마이크보다 인식 정확도가 높습니다.

발음 연습 화면에서 '내 스크립트' 탭을 선택하고, 발표문, 영어 에세이, 면접 답변 등 연습하고 싶은 영어 텍스트를 붙여넣은 뒤 '연습 시작'을 클릭하세요.

텍스트는 문장 단위로 자동 분리되어 순서대로 발음 연습이 진행됩니다. 모든 문장이 끝나면 문장별 점수와 전체 평균 점수가 요약 화면에 표시됩니다.

영어 문장만 인식되며, 한국어나 다른 언어가 섞인 경우 해당 줄은 자동으로 제외됩니다.

언어학 연구에 따르면 발음 교정은 짧게 자주 연습하는 것이 효과적입니다. 하루 30분 몰아서 하는 것보다 하루 10분씩 3회 나눠서 연습하는 방식이 기억 고착에 유리합니다.

CODEPRON 권장 루틴: 아침 5분(기본 연습 3~4문장) + 저녁 10분(스크립트 연습 또는 가이드 복습). 2~3주 꾸준히 실천하면 명확한 변화를 느낄 수 있습니다.

특히 틀린 단어를 집중 반복하는 것이 가장 효과적입니다. 결과 화면의 빨간색 단어를 5회 이상 집중 연습하세요.

네, 입문 레벨 문장은 영어를 처음 배우는 분도 도전할 수 있도록 간단하게 구성되어 있습니다. 영어 알파벳을 읽을 수 있다면 누구든 사용 가능합니다.

어린이의 경우 음성 인식 정확도가 성인에 비해 낮을 수 있으나(아동 음성 특성상), 발음을 소리 내어 연습하는 행위 자체가 학습에 도움이 됩니다. 부모와 함께 사용하는 것을 권장합니다.
